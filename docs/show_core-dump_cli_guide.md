### show core-dump

#### Syntax
``` show core-dump ```

#### Description
This command will list out all the core dumps present in the switch.

#### Authority
all users.

#### Parameters
No Parameters

#### Examples

```
switch#show core-dump

---------------------------------------------------
TimeStamp                  Core Dump Name
---------------------------------------------------
2016-01-20:14:17:15        ops-lldpd
```
